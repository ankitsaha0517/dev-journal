# Learning Log 001
 
Day 1 of the JavaScript fundamentals track in the cohort curriculum. Captures first hands-on contact with core language mechanics rather than notes-only theory.
 
`script.js` is a working scratchpad, not a demo app — each section is a runnable, commented experiment: variable declarations (`var` vs `let` vs `const`, and why `var`'s lack of block scope is a trap), the three `console` severity levels, arithmetic/increment operators, and the three browser dialog functions (`alert`, `confirm`, `prompt`). `index.html` exists only to load the script in a browser so `console` output and dialogs are actually observable.
 
Later day-numbered logs should follow the same pattern: one topic cluster per log, code kept runnable and commented in place rather than summarized separately, so the log doubles as both notes and proof-of-practice.
