# 📚 Learning Log #010

## 🎯 Topic
Introduction to the DOM — selecting elements, manipulating content/style, and event listeners

---

## 📝 Overview
This log covers the basics of interacting with the DOM using vanilla JavaScript. It moves from selecting elements on the page to changing what they display, how they look, and how they respond to user interaction — the core loop behind any interactive front-end.

---

## 📖 What I Learned
- Selected elements using `getElementById` and `querySelectorAll`
- Updated content with `innerHTML`, and removed a list item with `.remove()`
- Changed styles directly via the `.style` property (background, size, border-radius)
- Added a `click` event listener to a button using `addEventListener`
- Compared `innerHTML` vs `textContent` — one renders markup, the other displays it as plain text

Practiced against a small `index.html` with headings, paragraphs, a button, a box, and a list, since DOM work needs real elements to select and mutate.
