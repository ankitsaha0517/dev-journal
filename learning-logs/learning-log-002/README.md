# Learning Log 002
 
Day 2 of the JavaScript fundamentals track: conditionals and the `while` loop, moved from syntax into decision-making logic.
 
`script.js` works through nested `if` (login check with three dependent conditions, loan eligibility with two), then `if / else if / else` as the flatter alternative once branches stop nesting cleanly (grade calculator), then `while` for repetition (printing a range, accumulating a sum). The progression is deliberate: nested `if` first to see why deep nesting gets hard to read, then chained `else if` as the fix, then loops once repetition — not just branching — is needed. `index.html` only exists to load the script so `console.log` output is visible.
 
Later logs continue the pattern: each numbered log is one topic cluster, practiced as small runnable snippets in place rather than described separately.
